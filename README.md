# File manager written in MMBasic, designed for PicoCalc. 
 
HOW TO USE :

- up/down arrow to choose file
- left/right arrow to change page
- enter to execute ( run file ) / change dir
- 'd' to view directories
- DEL/r button to remove files
- 'q' to exit the program

DIRECTORY : 

- 'q' to go back ( exit from directories )
- Rest is almost the same like in the files

There is a MENU with all of the avaible commands

# You can change how many files will appear
# on a single page

- EDIT fileExp.bas 
- FIND 'CONTROL CENTER' at start
- CHANGE 'fileLimit'

NOTE: 6 is recommended for PicoCalc.
        If you want to make the number
        higher, you will need to manually 
        change the position of the page
        counter.
