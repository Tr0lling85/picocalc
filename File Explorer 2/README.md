# Graphical file explorer desingned for PicoCalc
# written in MMBASIC ( on PicoCalc )

How to use:

- use arrows to navigate
- "." / "," next/previous page
- 'Back' go back one level ( chdir ".." )
- 'Del' to delete files ( and dir if empty )

- 'Enter' execute 
-- ( .bas / .bas.bak / .txt / .wav / .bmp )

- "n" rename files/dir
- "k" make directory
- "i" info ( press again for more info )
- 'F1' to close make/rename windows
- 'Tab' change drives

- "h" help menu ( control keys )

- 'Shift'+'Esc' to terminate program
