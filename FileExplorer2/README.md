# Graphical file explorer desingned for PicoCalc
# written in MMBASIC ( on PicoCalc )

<img src="https://github.com/Tr0lling85/picocalc/blob/main/FileExplorer2/img1.png">

## **How to use:**

- use arrows to navigate
- "." / "," next/previous page
- 'Back' go back one level ( chdir ".." )
- 'Del' to delete files ( and dir if empty )

- 'Enter' execute ( .bas / .bas.bak / .txt / .wav / .bmp )

    <br>

    **bas** or **bas.bak**
    - run file
    - NOTE : after running a file, you can't go back
      to the explorer, because PicoCalc cannot multitask

    **txt** 
    - set character limit ( characters per line )
    - set line limit ( lines per page )
    <br>

    <img src="https://github.com/Tr0lling85/picocalc/blob/main/FileExplorer2/img4.png">

    **wav**
    - Pause - pause sound
    - Play - resume sound
    - Play Again - play sound again
    - Close - close sound window

    **bmp**
   - press anything to close image
   <br>
   
- "n" rename files/dir
- "k" make directory
- "i" info ( press again for more info )

**info shows**
- name, size and date of the file when it was moddified

**pressing again**
- free space left on the drive ( in bytes )
- PicoCalc bootcount
- unique PicoCalc ID
<br> 

<img src="https://github.com/Tr0lling85/picocalc/blob/main/FileExplorer2/img2.png">

- 'F1' to close make/rename windows
- 'Tab' change drives ( A: / B: )

- "h" help menu ( control keys )
<br>

<img src="https://github.com/Tr0lling85/picocalc/blob/main/FileExplorer2/img3.png">

- 'Shift'+'Esc' to terminate program
